Installation Guide

Step 1: Boot the Live USB

1 Write the ISO to a USB drive using Rufus (Windows), Etcher (Mac/Linux), or dd command.
2 Boot from the USB drive. You'll see the XFCE desktop environment load automatically.

Step 2: Connect to the Internet

3 Open a terminal and run: systemctl start NetworkManager
4 Then run: nmtui to connect to WiFi using the text interface.

Step 3: Start the Installer

5 In the terminal (you're already logged in as root), type: archinstall
6 Follow the guided installation process. The installer will walk you through:
  • Selecting your disk and partitioning
  • Choosing your timezone and locale
  • Creating your user account
  • Installing the bootloader
⚠️ Important: Make sure to select XFCE as your desktop environment during installation to match the live environment you're using!

Step 4: Reboot and Enjoy

7 Once installation is complete, remove the USB drive and reboot. Your new Arch Linux system with XFCE is ready!
